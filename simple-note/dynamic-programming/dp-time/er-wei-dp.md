# 二維 DP

## 二維DP

其實二維 DP 並不是什麼特別的東西，就是在二維陣列上做 DP。會特別拉出來講是因為我很不會這種類型的 DP。有時候會忘記在二維陣列上面可以做 DP 這件事情。
